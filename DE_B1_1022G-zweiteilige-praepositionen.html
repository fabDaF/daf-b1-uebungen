📖 Karim plant seine erste Woche

Von morgen an lerne ich jeden Tag zehn neue Wörter, denkt Karim.

Das Konzert war bis auf den letzten Platz ausverkauft.

Von mir aus können wir auch zu Hause bleiben.

Er muss die Prüfung innerhalb von drei Monaten ablegen.

Auf deine Empfehlung hin habe ich den Fernseher gekauft.

Inmitten von vielen Aufgaben behält er den Überblick.

Was ist besonders an den markierten Ausdrücken?

Welche Präposition zeigt eine zeitliche Grenze ohne Ausnahme?

„Bis auf den letzten Platz ausverkauft" — was bedeutet bis auf hier?

Welchen Kasus verlangen die meisten zweiteiligen Präpositionen?

✅ Du hast die Regel entdeckt!

Zweiteilige Präpositionen bestehen aus zwei Wörtern, die zusammen eine Präposition ergeben.

Die meisten verlangen den Dativ. Nur bis auf und auf … hin verlangen den Akkusativ.

Dativ: innerhalb von einem Monat · von dem Fenster aus
Akkusativ: bis auf den letzten Platz · auf seine Bitte hin
📋
Zweiteilige Präpositionen bestehen aus zwei Teilen, die im Satz getrennt stehen können. Sie sind feste Ausdrücke.
Präposition Kasus Bedeutung Beispiel
bis auf Akkusativ einschließlich / ausgenommen Das Konzert war bis auf den letzten Platz ausverkauft.
auf … hin Akkusativ als Reaktion auf etwas Auf deine Empfehlung hin habe ich den Film gesehen.
bis zu Dativ zeitlicher / räumlicher Endpunkt Es sind 100 Meter bis zur Bushaltestelle.
inmitten von Dativ in der Mitte von etwas Das Hotel liegt inmitten von Wäldern.
aufgrund von Dativ wegen, aus dem Grund Aufgrund von Bauarbeiten ist die Straße gesperrt.
innerhalb von Dativ in einer Zeitspanne / im Bereich Er muss innerhalb von drei Tagen antworten.
von … aus Dativ Ausgangspunkt (räumlich/persönlich) Von mir aus können wir sofort gehen.
von … an Dativ Beginn eines Zeitraums Von morgen an spare ich jeden Monat.
außerhalb von Dativ außerhalb eines Bereichs Sie wohnt außerhalb von der Stadt.
gegenüber von Dativ auf der anderen Seite Das Café ist gegenüber von der Bank.

💡 Merkhilfe

Nur zwei verlangen den Akkusativ: bis auf und auf … hin. Alle anderen verlangen den Dativ.

Bei von … aus gibt es zwei Bedeutungen: „Von mir aus" = meinetwegen / „Vom Fenster aus" = vom Standpunkt des Fensters.
