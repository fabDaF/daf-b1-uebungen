🔍 Entdecken: Was bedeutet lassen?
Lies die Sätze. Was bedeutet lassen in jedem Satz? Überlege vor dem Weiterlesen!

Gruppe A — Auftrag / Kausativ

Ich lasse mein Auto reparieren.
→ jemand anderes repariert es für mich
Sie lässt sich die Haare schneiden.
→ beim Friseur
Wir lassen das Haus streichen.
→ Maler macht es

Gruppe B — Erlaubnis / Zulassen

Die Eltern lassen das Kind fernsehen.
→ sie erlauben es
Er lässt mich das Buch behalten.
→ er erlaubt, dass ich es behalte
Lass mich das erklären!
→ gib mir die Möglichkeit

Gruppe C — Vollverb (zurücklassen / liegenlassen)

Er lässt die Tür offen.
→ er schließt sie nicht
Ich habe den Schlüssel zu Hause gelassen.
→ nicht mitgenommen
Lass das!
→ hör damit auf!

Gruppe D — Bewirken / Ermöglichen

Das lässt sich nicht ändern.
→ es ist nicht möglich
Der Text lässt sich gut lesen.
→ man kann ihn gut lesen
Das Problem lässt sich lösen.
→ es ist lösbar

💡 Beobachtung: Was haben alle Sätze gemeinsam?

In Gruppe A und B folgt lassen einem Infinitiv → lassen ist hier ein Hilfsverb (Modalverb-ähnlich).
In Gruppe C steht lassen allein → es ist ein Vollverb.
In Gruppe D folgt sich + Infinitiv → das drückt eine Möglichkeit oder Passivbedeutung aus.

Die 5 Bedeutungen von lassen

1. Auftrag / Kausativ
to have something done
Ich lasse das Auto reparieren.
(Ein Mechaniker repariert es für mich.)
2. Erlaubnis / Zulassen
to let / to allow
Die Eltern lassen das Kind spielen.
(Sie erlauben es.)
3. Bewirken / Ermöglichen
to make / to cause
Das Licht lässt den Raum größer wirken.
(Es bewirkt diesen Effekt.)
4. Möglichkeit (sich lassen)
can be done / is possible
Das Problem lässt sich lösen.
(= Das Problem kann gelöst werden.)
5. Vollverb: zurücklassen / unterlassen
to leave / to stop
Er hat den Schlüssel zu Hause gelassen.  |  Lass das! (= Hör auf!)

⚠️ Das Perfekt von lassen — zwei Formen!

Als Vollverb:
Ich habe den Schlüssel hier gelassen.  (hat + gelassen)
Als Hilfsverb:
Ich habe das Auto reparieren lassen.  (hat + Infinitiv + lassen)
⚡ Als Hilfsverb steht lassen im Infinitiv (wie Modalverben: ich habe kommen müssen).
VerwendungTypStrukturBeispiel
Auftrag / Erlaubnis Hilfsverb lassen + Inf. Er lässt das Paket liefern.
Möglichkeit Hilfsverb sich lassen + Inf. Das lässt sich erklären.
Zurücklassen / aufhören Vollverb lassen allein Er lässt die Tür offen.
